    Bem vindo à página do That Piano Program.

    Este programa decorre da disciplina de Técnicas de Programação (Turma II) do curso de Engenharia Elétrica da UFCG, no período 2013.2, ministrado pelo professor Marcus Salerno. Os autores deste programa, por ordem alfabética, são:

    1. Hugo Gayoso Meira Suassuna de Medeiros
    2. Fernando Marreiro II
    3. Matheus Alves dos Santos
    4. Matheus de Araújo Cavalcante (mantenedor)

    Instalação

    Instalação no Windows

    Este programa foi compilado utilizando o Visual Studio Ultimate 2012. Para a execução do That Piano Program é essencial, portanto, a instalação do Visual C++ Redistributable for Visual Studio 2012 Update 4, versão x86. Ele pode ser encontrado aqui.

    Após isso o download do executável pode ser realizado. A última versão estável do That Piano Program e pode ser encontrada na página de releases, aqui.

    Instalação no Linux

    Uma PPA está sendo criada. Infelizmente esta opção não está disponível nesta versão.

    Suporte

    Algum problema com o That Piano Program? Envie um e-mail para o mantenedor (matheus.cavalcante@ee.ufcg.edu.br) e ele terá prazer em respondê-lo.
