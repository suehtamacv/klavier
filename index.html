    Bem vindo à página do Klavier.

    Este programa decorre da disciplina de Técnicas de Programação (Turma II) do curso de Engenharia Elétrica da UFCG, no período 2013.2, ministrado pelo professor Marcus Salerno. Os autores deste programa, por ordem alfabética, são:

    1. Fernando Marreiro II
    2. Hugo Gayoso Meira Suassuna de Medeiros
    3. Matheus Alves dos Santos
    4. Matheus de Araújo Cavalcante (mantenedor)

    Instalação

    Este programa foi compilado utilizando o Visual Studio Ultimate 2012. Para a execução do Klavier é essencial, portanto, a instalação do Visual C++ Redistributable for Visual Studio 2012 Update 4, versão x86. Ele pode ser encontrado aqui.

    Após isso o download do executável pode ser realizado. A última versão estável do Klavier pode ser encontrada na página de releases, aqui.

    Suporte

    Algum problema com o Klavier? Envie um e-mail para o mantenedor (matheus.cavalcante@ee.ufcg.edu.br) e ele terá prazer em respondê-lo.
